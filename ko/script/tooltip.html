Tooltip

툴팁은 상/하/좌/우 총 4가지 위치로 설정할 수 있으며 관련 메시지는 title 속성에 입력하면 됩니다. 

<span class="tooltip" title="Tooltip Message">Demo</span>

Top

툴팁의 위치를 상단으로 설정할 수 있으며 기본적으로 마우스 오버시에 툴팁이 보여집니다. (Over) 

jui.ready([ "ui.tooltip" ], function(tooltip) {
	tooltip_1 = tooltip(".tooltip_1");
});

Bottom

툴팁의 위치를 하단으로 설정할 수 있으며 컬러는 회색으로 툴팁은 마우스 클릭시에 보여집니다. (Click) 

tooltip_2 = tooltip(".tooltip_2", {
	position: "bottom",
	color: "gray",
    showType: "click"
});

Left

툴팁의 위치를 좌측으로 설정할 수 있으며 마우스 오버시 1초 후에 툴팁이 보여집니다. (Delay) 

tooltip_3 = tooltip(".tooltip_3", {
	position: "left",
	delay: 1000
});

Right

툴팁의 위치를 우측으로 설정할 수 있으며 max-width 값을 300px로 정합니다. 만약에 툴팁이 길어지면 자동으로 줄바꿈을 합니다.
그리고 툴팁은 가운데 정렬됩니다. (Over) 

tooltip_4 = tooltip(".tooltip_4", {
	position: "right",
	width: 300,
	align: "center"
});

With Popover

툴팁 스타일만이 아닌 팝오버 스타일과도 함께 사용할 수 있습니다. (Over) 

tooltip_5 = tooltip(".tooltip_5", {
    position: "right",
    width: 300,
    align: "left",
    tpl: {
        item: $("#tpl_popover").html()
    }
});

팝오버 스타일을 사용하기 위해서는 item 템플릿을 설정해야 합니다. 

<script id="tpl_popover" type="text/template">
    <div class="popover popover-<!= position !> popover-<!= color !>">
        <div class="head">Subject</div>
        <div class="body">
            <div class="image"><i class="icon-caution"></i></div>
            <div class="message"><!= message !></div>
        </div>
    </div>
</script>
