Row

Properties

It is possible to directly refer to a row object and row.data in a row template.

Name Description
type State value that indicates whether a child row is shown or hidden.
data Data of a specified row
rownum The unique number of a child row under the specified parent row if a parent row exists.
index Index of a specified row. In the case of a tree structure, a depth is given.
element TR element of a specified row
list List of TD elements of a specified row
parent Variable that refers to the parent row
children List of child rows.
depth The depth of the current row in the case of a tree structure
