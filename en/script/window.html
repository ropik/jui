Window

Default

The window is a layer component that can replace pop-ups.

Result Description
Window for which relocation and resizing is available (Black & White)
Window for which relocation and resizing is available (White)
Window for which relocation and resizing is available (Black)
Window that is shown in a modal form

The following is window-related markup code and a script for creating a UI. 

<div id="win_1" class="window">
	<div class="head">
		<div class="left">HOME</div>
		<div class="right">
			<a href="javascript:alert('plus');"><i class="icon-plus icon-white"></i></a> 
			<a href="javascript:alert('search');"><i class="icon-search icon-white"></i></a> 
			<a href="#" class='close'><i class='icon-exit icon-white'></i></a>
		</div>
    </div>
	<div class="body">
		Contents...<br/>
	</div>
</div>

jui.ready([ "uix.window" ], function(window) {
	win_1 = window("#win_1", { 
		width: 500, 
		height: 300, 
		left: 100, 
		top: 100
	});
	
	win_1_2 = window("#win_1_2", { 
		width: 500, 
		height: 300, 
		left: 100, 
		top: 400
	});
	
	win_1_3 = window("#win_1_3", { 
		width: 500, 
		height: 300, 
		left: 100, 
		top: 700
	});
	
	win_2 = window("#win_2", { 
		width: 500, 
		height: 300, 
		modal: true
	});
});
