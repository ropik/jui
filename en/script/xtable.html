Large table which can process millions of data rows.
The inner scroll problem observed in the existing performance has been resolved and a buffer-based paging or scrolling function is provided.

jui.ready([ "uix.xtable" ], function(xtable) {
	xtable_1 = xtable("#xtable_1", {
		fields: [ "name", "age", "location" ],
		resize: true,
		colshow: true,
		sort: true,
		sortLoading: true,
		buffer: "scroll", // scroll, page, s-page
		bufferCount: 20,
		event: {
			colmenu: function(column, e) {
				this.showColumnMenu(e.pageX - 25);
			}
		}
	});
});

Filter

Filter Multiple Filter - When clicking the button on the left side, the rows of a table are filtered according to specific criteria.

Name Age Location 

function xtableFilter() {
    xtable_4.filter(function(data) {
        if(data.location.indexOf("eo") != -1) {
            return true;
        }
    });
}

function xtableFilterMulti() {
    xtable_4.filter(function(data) {
        if(data.age >= 30 || data.name.indexOf("ng") != -1) {
            return true;
        }
    });
}

Horizontal Scroll

If the number of columns is big or the table width is large, the use of the scrollWidth option enhances the table legibility.

Name Age Location

The horizontal scroll is shown If the actual area of a table is greater than the scrollWidth option.
The actual area of a table can be set using the width option or the style of a markup. 

xtable_5 = xtable("#xtable_5", {
	fields: [ "name", "age", "location" ],
	data: table_data,
	resize: true,
	sort: true,
	width: 1000,
	scrollWidth: 727,
    animate: true
});
