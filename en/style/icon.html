Icon

Default

The icon component has the same properties as font, and the user can select the size and the color of an icon using the code shown below.   

<i class="icon-gear" style="font-size: 12px; color: red;">
<i class="icon-gear" style="font-size: 15px; color: blue;">
<i class="icon-gear" style="font-size: 18px; color: green;">
<i class="icon-gear" style="font-size: 21px; color: black;">

Animation

The icon component can optionally use various animation effects. 

<i class="icon-gear icon-spin"></i>
<i class="icon-gear icon-flash"></i>
<i class="icon-gear icon-bounce"></i>
<i class="icon-gear icon-float"></i>
<i class="icon-gear icon-pulse"></i>
