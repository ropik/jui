Vertical Group

Default

This style can be used to group each component in a vertical form. It has similar features to the Group style.
However, a Vertical Group, unlike a Group, provides functions for aligning elements. 

<div class="vgroup">
	<button class="btn btn-gray">Vertical Group <i class="icon-gear" style="float: right; margin-top: 7px;"></i></button>
	<button class="btn btn-gray">Vertical Group <i class="icon-gear" style="float: right; margin-top: 7px;"></i></button>
	<button class="btn btn-gray">Vertical Group <i class="icon-gear" style="float: right; margin-top: 7px;"></i></button>
</div>
